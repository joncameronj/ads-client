Duplicate a winning ad into a different ad set. Reads the source ad's creative and creates a new ad in the target ad set.

Use the `mcp__meta-ads__duplicate_ad` tool.

Required from user:
- source_ad_id: Ad ID to duplicate
- target_adset_id: Ad set ID to create the duplicate in

Optional:
- new_name: Name for the new ad (defaults to "[original name] - Copy")

Confirm the duplication and show the new ad ID.
