Create a new Meta ad campaign. Created in PAUSED status by default for safety.

Use the `mcp__meta-ads__create_campaign` tool.

Required from user:
- name: Campaign name
- objective: OUTCOME_LEADS, OUTCOME_TRAFFIC, OUTCOME_SALES, OUTCOME_AWARENESS, OUTCOME_ENGAGEMENT, or OUTCOME_APP_PROMOTION

Optional:
- buying_type: AUCTION (default) or RESERVED
- special_ad_categories: e.g., HOUSING, EMPLOYMENT, CREDIT

Confirm the campaign was created and show the new campaign ID.
